## <a href="https://github.com/ZhaoTongFeng/HiSpider/blob/master/Document/WebStruction.md">设计思路文档</a>

## **主要功能：**
- 提取任意文章页标题、作者、时间、点赞、评论和正文等
- 站点首页各个模块信息，如版块名、内容链接等
- 支持静动态，AJAX爬取
- 多页面爬取，大数据分析
- 验证，登录，上传等
- 自定义脚本，定时爬取，发送邮件报告，监控网站、签到等
- 代理池管理系统
